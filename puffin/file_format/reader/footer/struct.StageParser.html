struct StageParser {
    stage: ParseStage,
    file_size: u64,
    flags: Flags,
    payload_size: u64,
    metadata: Option<FileMetadata>,
}
Expand description

Manages the parsing process of the file’s footer.

Fields§

§stage: ParseStage

Current stage in the parsing sequence of the footer.

§file_size: u64

Total file size; used for calculating offsets to read from.

§flags: Flags

Flags from the footer, set when the Flags field is parsed.

§payload_size: u64

Size of the footer’s payload, set when the PayloadSize is parsed.

§metadata: Option<FileMetadata>

Metadata from the footer’s payload, set when the Payload is parsed.

fn next_to_read(&self) -> Option<BytesToRead>

Determines the next segment of bytes to read based on the current parsing stage. This method returns information like the offset and size of the next read, or None if parsing is complete.

source

fn consume_bytes(&mut self, bytes: &[u8]) -> Result<()>

Processes the bytes that have been read according to the current parsing stage and advances the parsing stage. It ensures the correct sequence of bytes is encountered and stores the necessary information in the StageParser.

source

fn finish(self) -> Result<FileMetadata>

Finalizes the parsing process, ensuring all stages are complete, and returns the parsed FileMetadata. It converts the raw footer payload into structured data.
