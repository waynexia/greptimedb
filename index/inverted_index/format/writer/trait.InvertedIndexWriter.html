where Self: 'async_trait, 'life0: 'async_trait,

Adds entries to an index.

  • name is the index identifier.
  • null_bitmap marks positions of null entries.
  • values is a stream of values and their locations, yielded lexicographically. Errors occur if the values are out of order.
